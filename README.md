# Mobile_Sales_Power_BI_DAX_Formula

## Project Overview

This project is an end-to-end example for beginners learning Power BI.  
An interactive and mobile-friendly Sales Analysis Dashboard was created using DAX formulas.  
Through this project, users can gain hands-on experience with data modeling, creating measures, visualizations, and report sharing in Power BI.

---

## DAX Measures / KPIs Used

| Measure Name | Description |
|-------------|-------------|
| Total Sales | Calculates the total sales amount |
| Total Quantity | Shows the total quantity of products sold |
| Average Price | Calculates the average sales price |
| Transaction Count | Counts the total number of transactions |
| Rating Status | Classifies product or customer ratings |
| YTD (Year-To-Date) | Calculates sales from the start of the year until today |
| MTD (Month-To-Date) | Shows sales from the beginning of the month until today |
| Same Period Last Year (SPLY) | Compares sales with the same period last year |

---

## Time-Based Performance Analysis

Time-based DAX functions were used in the dashboard:

- **MTD (Month-to-Date)**  
  Shows sales from the beginning of the current month until today.  
  *Purpose:* To monitor current month performance and compare with previous months.

- **QTD (Quarter-to-Date)**  
  Calculates data from the beginning of the current quarter until today.  
  *Purpose:* To analyze progress toward quarterly targets.

- **YTD (Year-to-Date)**  
  Shows total sales or revenue from the start of the year until today.  
  *Purpose:* To evaluate achievement of annual goals and growth trends.

- **SPLY (Same Period Last Year)**  
  Compares data with the same period last year.  
  *Purpose:* To measure year-over-year performance differences and trends.
